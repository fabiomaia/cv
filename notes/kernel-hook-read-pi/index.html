Hook read syscall to find pi

I like pi, and I wanted to learn more about kernel modules, so I decided to write a kernel module that hooks the read syscall and print a message if it finds the string "pi" in the buffer that was read out.

echo "I like pi" > foo.txt
insmod pimod.ko
cat foo.txt
dmesg | grep pi
