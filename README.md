# fabiomaia.pt

## Install

### Repository

```shell
$ git clone git@github.com:faviouz/fabiomaia.pt.git
$ cd fabiomaia.pt/
```

### Development Environment

```shell
$ vagrant up
$ vagrant ssh
$ cd /vagrant
```

### Dependencies

```shell
$ npm install
```

## Build

```shell
$ gulp
```

## Deploy

```shell
$ gulp deploy
```
